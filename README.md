# TermBrain: Privacy-First AI CLI Diagnostic tool

A CLI built with Python, Typer, and Rich, leveraging Ollama for local LLM-powered system diagnostics.

## Development

1.  **Clone the repo** (if applicable)
2.  **Install locally in editable mode**:
    ```bash
    pip install -e .
    ```
3.  **Run TermBrain**:
    ```bash
    termbrain
    ```
